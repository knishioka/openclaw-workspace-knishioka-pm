# Learning Recommendations (2026-03-28)

## 現在の推奨設定

### kanji-practice
- 学年: **2年生** (漢検9級対策)
- 問題タイプ: **書き練習** (読み→書きのステップで進める)
- ランダム: ON
- ページ数: 1 (多ページ生成バグ Issue #20 が修正されるまで)
- 備考: 9級の読みが8割定着したら、書き練習の比率をさらに上げる

### math-worksheet
- 学年: **4年生** (3年生には3桁×1桁がないため)
- 問題タイプ: **3桁×1桁のかけ算**
- 列: 3列, 30問
- 備考: 安定したら割り算の筆算や3桁×2桁に移行

### english-note-maker
- モード: **フレーズ練習** (あいさつ)
- 年齢: **7-9歳** (10-12歳は現段階では難しすぎる)
- 罫線: 10mm
- ページ数: 2
- 備考: 写経が安定したら穴埋めモードへ (Issue #19 が実装されたら)

## 次のステップ
- kanji: 9級(小2)漢字160字の読みが8割定着 → 書き練習中心に切り替え → 8級(小3)漢字を先行導入
- math: 3桁×1桁が安定 → 割り算の筆算 → 3桁×2桁のかけ算
- english: 写経の安定 → 穴埋め (Issue #19) → 短文の自由記述

## 気づき
- math-worksheet は対象児童の算数レベル(小3-4相当)に対して、学年選択では4年生を選ぶ必要がある
- english-note-maker の「文章練習/10-12歳」は語彙が抽象的で、現段階の対象児童には負荷が高い
